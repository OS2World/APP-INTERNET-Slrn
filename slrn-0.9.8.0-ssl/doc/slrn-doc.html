Accessing slrn's online documentation
Last update: Tue Aug 19 23:45:36 CEST 2003
Information
The current documentation is also available online from http://slrn.sourceforge.net/docs/. Besides the plain text versions that come with the source distribution, this page offers HTML versions of many documents. The slrn reference manual and FAQ have their own page, http://slrn.sourceforge.net/manual/.
Where available, both pages also link translated versions of the docs; for additional (non-official) documentation and tutorials (both in English and in some other languages), look at the list at our links page.
